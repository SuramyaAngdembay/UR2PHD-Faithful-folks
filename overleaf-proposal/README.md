# Overleaf Proposal Starter

Upload these two files to the shared Overleaf project:

- `main.tex`
- `ur2phd.bib`

Set `main.tex` as the main document and click **Recompile**. The project uses
standard packages and should compile with Overleaf's default pdfLaTeX compiler.

Before submitting:

1. Confirm every group member is listed in the author line and team table.
2. Confirm `FaithCoT-Bench` is the paper your group used for Lesson 2. If a
   different paper was assigned, replace its BibTeX entry and citation.
3. Recompile twice so all citations and references appear.
4. Download the compiled PDF and submit one copy for the entire group.
